###  🔀 작업가이드

1️⃣ 사용자 입력 처리
- 이름, 나이, 이메일, 연락처를 입력받기
- 나이는 정수형으로 형 변환 (`int()`)
- 이메일은 '@'를 기준으로 ID와 도메인 분리 처리 (`split("@")`)
- 연락처는 문자열 그대로 입력받고 나중에 포맷 가공

2️⃣ 데이터 가공
- 이름을 모두 대문자로 변환 (`upper()`)
- 이름 길이 구하기 (`len()`)
- 이름의 첫 글자 추출 (`name[0]`)
- 나이에 5를 더한 값 계산 (`user_age + 5`)
- 나이 제곱값 계산 (`user_age ** 2`)
- 나이를 4로 나눈 몫과 나머지 계산 (`//`, `%`)
- 이메일 ID 길이 계산 (`len(id_part)`)
- 연락처 문자열에서 국번과 개인번호 추출 (`split("-")` 또는 인덱싱)

---
3️⃣ 데이터 검증 및 예외 처리
- 나이 입력이 숫자가 아닐 경우 `ValueError` 처리
- 이메일 입력에 '@'가 없을 경우 `IndexError` 처리
---
4️⃣ 리포트 출력 구성
- 구분선(`=====`) 삽입
- 이름/나이/취미/이메일/연락처 요약 출력
- 분석 결과(문자열 길이, 도메인, 나이 계산 결과 등) 포함
- 사용자 친화적 메시지 추가 (예: "데이터 분석 완료!")

5️⃣ 출력 포맷 구성
- 출력 내용은 다음 양식을 참고하여 구성할 것:

```python
# 사용자 입력 처리
user_name = input("이름을 입력하세요: ")
user_age_input = input("나이를 입력하세요: ")
user_email = input("이메일을 입력하세요: ")
user_phone = input("연락처를 입력하세요 (예: 010-1234-5678): ")

# 예외 처리: 나이 입력이 숫자가 아닐 경우 처리
try:
    user_age = int(user_age_input)  # 나이 숫자로 변환
except ValueError:
    print("❗ 나이는 숫자로 입력해야 합니다.")
    user_age = 0  # 기본값 설정

# 데이터 가공

# 이름 처리
user_name_upper = user_name.upper() # 대문자로 변환
name_length = len(user_name_upper)  # 이름 길이
first_letter = user_name_upper[0] # 이름 첫 글자 추출

# 나이 처리
after_five_years = user_age + 5  # 5년 후 나이
age_squared = user_age ** 2   # 나이 제곱
age_quotient = user_age // 4  # 나이 나누기 4 몫
age_remainder = user_age % 4 # 나이 나누기 4 나머지

# 이메일 처리
email_contains_at = "@" in user_email # @ 포함 여부
try:
    id_part, domain_part = user_email.split("@")  
    # 이메일 ID와 도메인 분리
    
    id_length = len(id_part) # ID 길이
except ValueError:
    domain_part = "없음"
    id_length = 0

# 연락처 처리
try:
    phone_parts = user_phone.split("-") # 하이픈으로 분리
    middle_number = phone_parts[1] # 국번
    last_number = phone_parts[2] # 개인 번호 끝자리
except IndexError:
    middle_number = "알수없음"
    last_number = "알수없음"

# 최종 출력

print("\n===== 사용자 정보 리포트 =====\n")

# 이름 출력
print(f"이름: {user_name_upper}")
print(f"이름 길이: {name_length}자")
print(f"첫 글자: {first_letter}\n")

# 나이 출력
print(f"나이: {user_age}살")
print(f"5년 후 나이: {after_five_years}살")
print(f"나이 제곱: {age_squared}")
print(f"나이를 4로 나눈 몫: {age_quotient}, 나머지: {age_remainder}\n")

# 이메일 출력
print(f"이메일 도메인: {domain_part}")
print(f"이메일에 '@' 포함 여부: {email_contains_at}")
print(f"이메일 ID 길이: {id_length}자\n")

# 연락처 출력
print(f"연락처: {user_phone}")
print(f"국번: {middle_number}")
print(f"개인번호 끝자리: {last_number}\n")

print("💡 데이터 분석 완료!")
print("============================")
```
