비용 줄이는 팁
1. 모델 등급 스위칭
    - 기본은 GPT-4o mini로 개발/테스트 → 진짜 필요한 경로만 상위 모델로 스위칭. 
        
2. 응답 길이 제어
    - 프롬프트에 최대 N토큰/짧게 등 제한을 주고, 로그로 평균 토큰을 모니터링.
        
3. 이미지 생성은 등급 조절
    - 시안 단계는 저/중 품질, 최종 산출만 고품질. 
        
4. STT는 묶음 처리
    - 너무 잘게 쪼개면 과금 비효율. 30~60초 단위 이상으로 배치 처리.
        
5. File Search는 1GB 무료를 적극 활용
    - 주간 PoC는 1GB 내 유지 + 필요 시 만 호출. 툴 호출도 배치/캐시
        
6. 월 한도 설정
    - OpenAI 대시보드에서 월별 예산·알림 설정(초과 청구 방지).
